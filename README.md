# AI 속에 숨어라 (Hide In AI)

AI NPC들 사이에 숨어서 AI인 척 행동하며, 다른 플레이어(사람)를 찾아 제거하는 1인칭 VR 라스트맨 스탠딩 게임.

## 게임 개요

- **플랫폼:** Meta Quest 2/3 (스탠드얼론)
- **플레이어:** 8~12명 + AI NPC 20~30명
- **라운드:** 3~5분
- **아트:** 로우폴리 카툰
- **네트워크:** Photon PUN2

## 핵심 규칙

1. 모든 캐릭터(사람 + AI)가 동일한 외형
2. 수상한 대상을 칼로 제거
3. AI를 3번 잘못 죽이면 본인 사망 (오킬 페널티)
4. 마지막 생존자가 승리

## 개발 환경

- Unity 2022.3 LTS (URP)
- Visual Studio
- XR Interaction Toolkit
- Photon PUN2
- Meta Quest 2/3
